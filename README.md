**Stack:** React.js 18 · Spring Boot 3.2 · MySQL 8 · JWT · Swagger/OpenAPI 3

---

## Prerequisites

| Tool | Version |
|------|---------|
| Java | 17+ |
| Maven | 3.8+ |
| Node.js| 18+ |
| MySQL | 8+ |

## Setup Instructions

### 1. Database
Create a MySQL database named `shopkart` and update `application.properties` with your MySQL username and password.

### 2. Backend
Navigate to the `backend` directory and run:
```bash
mvn spring-boot:run
```

### 3. Frontend
Navigate to the `frontend` directory, install dependencies, and start the app:
```bash
npm install
npm start
```
